# SOC Detection Labs

## Overview

This repository contains hands-on SOC analyst and detection engineering projects completed in a virtual cybersecurity lab environment.

## Lab Environment

- Windows Host
- VirtualBox
- Kali Linux (Attacker)
- Kali Linux (Victim)
- Splunk Enterprise
- Wireshark

## Projects

### SSH Brute Force Detection
- Simulated SSH brute-force attacks using Hydra
- Created Splunk alerts and dashboards
- Investigated authentication logs

### Port Scan Detection
- Performed Nmap reconnaissance scans
- Created SIEM detections for scan activity

### Reverse Shell Detection
- Generated reverse shell connections
- Analyzed suspicious outbound traffic

### End-to-End SOC Simulation
- Simulated reconnaissance, exploitation, and shell access
- Built incident timeline and investigation findings

## Tools Used

- Kali Linux

## Skills Demonstrated

- SIEM Monitoring
